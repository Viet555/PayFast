## 💸 Money Transfer Web Application

This project is a web-based money transfer system that allows users to securely send and receive funds between accounts.
The application simulates core banking functionalities such as user authentication, balance management, transaction processing, and transaction history tracking.

Built with Node.js, Express, Sequelize, and MySQL, the system follows the MVC architecture and ensures data consistency using database transactions.

The goal of this project is to demonstrate backend development skills, database design, and secure financial transaction handling.
